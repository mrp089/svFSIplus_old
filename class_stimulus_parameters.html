Detailed Description

The StimulusParameters class stores parameters for 'Stimulus' XML element used to parameters for pacemaker cells.

<Stimulus type="Istim" >
<Amplitude> -52.0 </Amplitude>
<Start_time> 0.0 </Start_time>
<Duration> 1.0 </Duration>
<Cycle_length> 10000.0 </Cycle_length>
</Stimulus>
